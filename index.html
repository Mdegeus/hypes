Docks

A dock is essentially a position fixed container wich can be used to show the user data on top of the actual page.
There are 2 types of dock: Both look the same but have another functionality.

Class: dock is triggered by a button press and can be set up in this way:

<div class="dock rounded shadow slide" id="dock"> <div class="dock-content"> <p class="dock-title center-text br">Title goes here</p> <p class="center-text br">You can add any element to the "dock-content" div.</p> <div class="dock-container"> <button class="dock-close" dock-id="dock">Close</button> </div> </div> </div>
Class: dock-alert is triggered on reload or when applied to existing dock with javascript. It can be made like this:

<div class="dock rounded shadow dock-alert slide" autoclose=2 id="warning"> <div class="dock-content"> <p class="dock-title">Dock title goes here</p> <p>Place any element here.</p> </div> </div>

Dock child elements

dock-title This can be applied to text elements. It will make the text bold and bigger then the rest of the text inside the dock.

dock-container wich is applied to a div shows all its containing elements with: display: space between;
This can be used to center a button or set 2 buttons in center with the same space around them. It works on other elements as well.

You can make a trigger button for a dock by adding the class dock-button and the target dock-id as a atribute
Here is an example:

<button class="dock-button" dock-id="dock">Open "dock"</button>
